这版怎么向 Linux.do 靠拢

配色一比一映射

直接使用你抓到的站点变量:链接与主操作 --tertiary #08c,点赞 --love #fa6c8d,已采纳 --success #090,高优先级角标 --danger #c80001,文字与灰阶 #222 / #646464 / #909090 / #e9e9e9,头部横幅渐变用 tertiary-low #d1efff。

Discourse 个人页版式

头部照搬 Web 端个人页:淡蓝横幅 + 大头像 + 用户名/头衔,下接「摘要 / 活动 / 徽章 / 偏好设置」横向页签(active 用 2px 蓝色下划线)。统计块即 Discourse 的「统计」区。

话题列表行

「最近活动」直接用论坛话题行的语法:标题 + 分类色块(开发调优蓝 / 搞七捻三橙红)+ 回复数 + 相对时间,被采纳的回复带绿色 ✓。这是最能让人一眼认出 Linux.do 的元素。

角标语义分级

沿用 Discourse 通知语义:常规未读(通知、草稿)用蓝色 #08c 角标,高优先级(私信)才用红色 #c80001——和网页端头像上的两种气泡一致。

圆角与密度

Discourse 的克制圆角:按钮/徽章 chip 4px,其余直角 + 1px #e9e9e9 分割线;行距和留白比 v2 略紧,更接近论坛的信息密度。

深色是推导的

你只发了浅色变量;深色版按同一语义推导(蓝提亮为 #33A6DD 等)。如果你在站点切到深色模式再跑一遍那段脚本发我,我就把深色也换成实测值。
